# Z Table Definitions

### ZMM_REQ_HEADER
| Field | Data Type | Description |
|--------|------------|-------------|
| REQ_ID | CHAR10 | Request ID |
| EQUIP_ID | CHAR10 | Equipment Reference |
| STATUS | CHAR10 | Current status |

### ZPM_ORDER_LOG
| Field | Type | Description |
|--------|------|-------------|
| ORDER_ID | CHAR10 | Maintenance order reference |
| CREATED_BY | CHAR12 | SAP user |
| PR_NO | CHAR10 | Purchase Requisition Number |
